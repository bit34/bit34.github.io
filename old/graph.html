Graph

This is an experimental graph library.

Features & Design Decisions:

  • No casting; Generic graph base classes needs to be extented for your own usage. So no cast is neccessary.
  • GridGraph; Most games with grid based data structures requires features like cell adjecency info, path finding. Using graph as a base class for grids is quite useful.
  • Rectangle grid graph
  • Hexagon grid graph
  • Basic path finding with djisktra algorithm. You can add heuristics to get Astar. Also supports partial paths.
