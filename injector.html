Injector

This is a simple dependency injection library with a small set of features. It is primarily developed to be used in conjunction with our other libraries. Its simple nature also makes it an easy point to start learning dependency injection.

If you do not know what dependency injection is and how it may improve your development processes here is a small tutorial.

Features & Design Decisions:

  • Only singleton bindings; every binded type to same type will share same instance.
  • Bind first, use later; after making first injection you can not make any more bindings and it will cause an error.
  • Compile time errors; generic classes will prevent binding not assignable type bindings
  • Runtime errors; errors that can not be detected by compile time will be informed through runtime error messages
