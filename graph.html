Graph

This is a minimal graph library.

Features & Design Decisions:

  • Graph base rectangle data; Most games with grid based data structures requires features like cell adjecency info, path finding so using graph as a base class for grids is quite useful.
  • No casting; Using generic base classes removes neccesity to cast graph elements
