Graph

This is a minimal graph library.

Features & Design Decisions:

  • Graph class with basic functionality
  • GridGraph (Graph child class) for regular structured graphs
  • RectGraph (GridGraph child class) for handling rectangular grids with all graph functionality
  • Agent class with basic path finding
