Package Manager

Unity does not support some neccesary features like packages with git dependencies. This tool will be a temporary solution until Unity implementes them.

Usage:

Add Bit34 Package Manager to tour project

  • You can open Unity Package Manager, click plus button, add this url to textbox
  • Or add following line directly to your Packages/manifest.json file

Add your packages to Assets/Bit34/repositories.json our file is given below

Add your dependencies to Assets/Bit34/dependencies.json, like in the example given below

br>

Everytime you modify the Assets/Bit34/dependencies.json press Reload button (and wait a little for changes to take affect)

To prevent your project's git interfering with libraries add this two lines to your .gitignore file

  • Assets/Bit34/Packages/
  • Assets/Bit34/Packages.meta

Time Manager

This is a minimal time library with scheduled callbacks. With this library you can;

  • Add callbacks to be called every application tick
  • Add callbacks to be called after given interval
  • Auto remove callbacks after given call count
  • Pause, resume and remove callback individualy or by their owner key
